Introducción




Muchas veces en la programación de una aplicación compleja se tienen que utilizar acciones o instrucciones repetitivas en el programa, lo cual hace muy tedioso para el usurario el hecho de volver y copiar la misma línea de codoco una y otra vez en el desarrollo de su aplicación, por esto en programación se generaron las funciones, estas están hecha con el objetivo de hacer el trabajo más fácil del trabajador y la reducción del código.


Las funciones son la solución a todos estos problemas, tanto en JavaScript como en el resto de lenguajes de programación. Una función es un conjunto de instrucciones que se agrupan para realizar una tarea concreta y que se pueden reutilizar fácilmente.


























Información




¿Qué es una función?

Porción de código reusable, al ser reusable se evita el estar escribiendo el mismo código una y otra vez.


Llamado de funciones

Para llamar una función desde cualquier parte de nuestro programa, solo es necesario escribir el nombre de la función creada seguido de paréntesis( en caso de tener parámetros indicarlos dentro, separados por comas).


Definición de una función

La manera más fácil de definir una función es con la palabra reservada "function" al cuál se le añade un único nombre de función y se decide si quiere llevar o no parámetros, junto con un bloque de instrucción que va dentro de llaves. Existen dos maneras de definir una función en JavaScript.


Declaración de Función

Se le asigna un nombre a la función. Dicha función se debe cargar en el ámbito de la aplicación antes de la ejecución del código.


Expresión de Función

Se crea una función anónima y se le asigna a una variable. Esta función no se encuentra lista hasta que esa línea en particular sea evaluada durante la ejecución del código.


Argumentos y valores de retorno

Las funciones más sencillas no necesitan ninguna información para producir sus resultados. Sin embargo, la mayoría de funciones de las aplicaciones reales deben acceder al valor de algunas variables para producir sus resultados.

La sentencia return finaliza la ejecución de la función y especifica un valor para ser devuelto a quien llama a la función.

Cuando una instrucción de retorno se llama en una función, se detiene la ejecución de esta. Si se especifica un valor dado, este se devuelve a quien llama a la función.

La instrucción de retorno se ve afectada por la inserción automática de punto y coma. No se permite el terminador de línea entre la palabra clave de retorno y la expresión.
